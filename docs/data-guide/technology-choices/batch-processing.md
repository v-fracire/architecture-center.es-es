<span data-ttu-id="12e83-103">Las soluciones de macrodatos a menudo utilizan trabajos por lotes de ejecución prolongada para filtrar, agregar y preparan de otra forma los datos para su análisis.</span><span class="sxs-lookup"><span data-stu-id="12e83-103">Big data solutions often use long-running batch jobs to filter, aggregate, and otherwise prepare the data for analysis.</span></span> <span data-ttu-id="12e83-104">Normalmente estos trabajos implican leer archivos de código fuente desde un almacenamiento escalable (por ejemplo, HDFS, Azure Data Lake Store, and Azure Storage), procesarlos y escribir la salida a los nuevos archivos de almacenamiento escalable.</span><span class="sxs-lookup"><span data-stu-id="12e83-104">Usually these jobs involve reading source files from scalable storage (like HDFS, Azure Data Lake Store, and Azure Storage), processing them, and writing the output to new files in scalable storage.</span></span>
